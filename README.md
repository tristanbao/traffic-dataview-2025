# 交通看·区域经济 2025

基于高速公路通行数据的浙江省区域经济分析可视化报告

## 项目简介

本项目通过可视化展示浙江省11个地市的经济活跃度、城际连接强度、产业流动性等多维度数据，为区域经济分析提供直观的数据支撑。

## 主要功能

- 📊 **区域经济活跃度指数** - 展示各地市经济活跃度与GDP对比
- 🌐 **城际连接强度网络** - 可视化城市间的连接关系
- 📈 **产业流动性指数** - 分析月度变化趋势
- ⭐ **多维度对比雷达图** - 对比重点城市的综合实力
- 🗺️ **浙江省热力分布图** - 地图展示区域经济分布

## 技术栈
